# Gato de Gatos / Ultimate Tic Tac Toe


Proyecto para la clase de Inteligencia Artificial de la Licenciatura en TICs de la ENES Morelia del semestre 2019-1.



\* Profesor: José de Jesús Pelayo Gómez.

\* Alumnos: Angélica Rivas, David Calderón, Jorge Camarena, Keshava Sanchez, Nikolai García y Stephany Vargas.



Esta es una implementación del algoritmo A\* en Python para un videojuego de jugador contra máquina. Este algoritmo garantiza siempre ganar o empatar, pero nunca perder.



\## A considerar


Proyecto hecho cuando todavía no aprendíamos buenas costumbres de versionado, por lo que no tenemos conocimiento de qué versiones de las librerías hacen que el código ejecute.
